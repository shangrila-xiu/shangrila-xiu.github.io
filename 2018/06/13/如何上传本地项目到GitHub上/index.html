两种上传本地到GitHub上方法

1、先登录自己的账号

2、创建仓库的步骤：

1、左键点击用户头像旁边的三角符号；

2、选择Your Profile，进入下一个页面；

3、选择这个页面下的Repositories，可以看到new按钮，并点击

4、进入下一个页面就可以来输入（Repository name）仓库的名称，下面是选择public，点击Create repository来创建仓库；

5、经过上面的步骤，仓库已经创建成功了。

6、接下来就是通过git命令来将本地的项目上传到github上。
