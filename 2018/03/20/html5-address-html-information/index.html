address联络信息

address元素不是html5新增的元素，但在定义上做了一些修改，用来标记距离它最近之父article元素或body元素的作者的联络信息。

address元素是专门用来标记网页或问还在那个的逝的联络信息。不是用来标记随便一个地址的，也不是用来标记其他非作者联络信息。

同一个网页可以包含多个article元素，而每个article元素可以包含各自的address元素，以标记改文章的作者的联络信息，有时address元素也会跟footer元素的其他信息放在一起。
